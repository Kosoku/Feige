ScopeFunctions

public protocol ScopeFunctions

Conformance to this protocol provides implementation of some of the Kotlin Scope Functions. To add conformance to your own types, see examples below.

class MyClass: ScopeFunctions {}

struct MyStruct: ScopeFunctions {}

See also

https://kotlinlang.org/docs/scope-functions.html

Public Functions

  • let(_:) Extension method

    Invokes the provided block and returns the result, providing the receiver as the only argument.

    struct MyStruct: ScopeFunctions {}

let retval = MyStruct().let {
   // do stuff with $0 and return a different value
   "my string value"
}

// do something with retval which is a String

    See also

    https://kotlinlang.org/api/latest/jvm/stdlib/kotlin/let.html

  • takeIf(_:) Extension method

    Invokes the provided predicate, if it returns true, returns the receiver, otherwise returns nil.

    struct MyStruct: ScopeFunctions {}

let retval = MyStruct().takeIf {
   // test $0 to determine return value
}

// do something with retval which is an optional MyStruct
